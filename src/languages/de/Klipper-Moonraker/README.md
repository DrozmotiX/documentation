## Klipper-Moonraker integration fur ioBroker

Ein ioBroker-Adapter zur Interaktion mit klipper über die Moonraker-API.

Der Testclient wurde mit Kiauh erstellt. Kiauh ist ein Skript, mit dem Sie eine perfekte Umgebung für Ihr Klipper-Setup erstellen können.
